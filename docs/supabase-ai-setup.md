# Supabase 로그와 무료 AI 확장 설정

## Supabase 이벤트 로그

정적 배포에서도 Supabase REST API로 사용 로그를 저장할 수 있습니다. 공개 저장소에는 Supabase service role key를 절대 넣지 말고, anon key와 RLS 정책만 사용하세요.

현재 클라이언트가 기록하는 이벤트는 다음과 같습니다.

- `search_input`: 검색어 입력
- `search_result_click`: 검색 결과 클릭
- `literature_tab_open`: 논문 근거 영역 열림
- `save_note_click`: 노트 저장 버튼 클릭
- `report_create_click`: Pro 리포트 생성 버튼 클릭
- `session_time`: 세션 체류 시간
- `learning_ai_expand_click`: AI 학습 확장 버튼 클릭

Supabase SQL Editor에서 아래 스키마를 실행하세요.

```sql
create table if not exists public.foldnote_events (
  id bigint generated by default as identity primary key,
  session_id text not null,
  event_type text not null,
  payload jsonb not null default '{}'::jsonb,
  occurred_at timestamptz not null,
  page_url text,
  user_agent text,
  created_at timestamptz not null default now()
);

alter table public.foldnote_events enable row level security;

create policy "allow public analytics insert"
on public.foldnote_events
for insert
to anon
with check (event_type in (
  'search_input',
  'search_result_click',
  'literature_tab_open',
  'save_note_click',
  'report_create_click',
  'session_time',
  'learning_ai_expand_click'
));
```

배포 페이지에서 아래 값을 주입하면 Supabase로 전송됩니다. 값이 없으면 브라우저 로컬 큐에만 쌓이고 사이트는 그대로 동작합니다.

GitHub Pages 같은 정적 호스팅에서 바로 쓰려면 `config.js`의 빈 값을 Supabase 값으로 채우세요. 브라우저에 공개되는 파일이므로 **service role key는 절대 넣지 말고 anon public key만** 넣어야 합니다. 로컬에서만 테스트하고 싶으면 아래 `localStorage` 방식도 사용할 수 있습니다.

```js
window.FOLDNOTE_SUPABASE = {
  url: "https://YOUR_PROJECT_ID.supabase.co",
  anonKey: "YOUR_SUPABASE_ANON_KEY"
};
```

브라우저 개발자 도구 콘솔에서 임시로 테스트하려면 아래처럼 넣은 뒤 새로고침해도 됩니다.

```js
localStorage.setItem("foldnote.supabaseUrl", "https://YOUR_PROJECT_ID.supabase.co");
localStorage.setItem("foldnote.supabaseAnonKey", "YOUR_SUPABASE_ANON_KEY");
location.reload();
```

## 로그 확인 쿼리

Supabase SQL Editor에서 아래 쿼리로 바로 확인할 수 있습니다.

```sql
-- 최근 이벤트
select
  created_at,
  event_type,
  payload->>'query' as query,
  payload->>'proteinName' as protein_name,
  payload->>'selectedId' as selected_id,
  payload->>'projectId' as project_id
from public.foldnote_events
order by created_at desc
limit 100;

-- 검색어 순위
select
  payload->>'query' as query,
  count(*) as count
from public.foldnote_events
where event_type = 'search_input'
group by query
order by count desc
limit 30;

-- 버튼/기능 사용량
select
  event_type,
  count(*) as count
from public.foldnote_events
group by event_type
order by count desc;

-- 평균 세션 시간
select
  round(avg((payload->>'durationMs')::numeric) / 1000, 1) as avg_seconds
from public.foldnote_events
where event_type = 'session_time';
```

## 무료 AI 학습 확장

학습 화면에는 AI 학습 확장 버튼이 들어 있습니다. 기본값은 외부 AI 키 없이 동작하는 로컬 학습 확장입니다. 무료 AI API를 붙이려면 브라우저에 개인 토큰을 직접 넣지 말고, Supabase Edge Function이나 다른 서버리스 함수 뒤에 토큰을 숨기는 구성을 권장합니다.

```html
<script>
  window.FOLDNOTE_AI = {
    endpoint: "https://YOUR_FUNCTION_URL",
    token: "",
    model: "free-model-name"
  };
</script>
```

AI 응답은 OpenAI 호환 `choices[0].message.content`, `generated_text`, `response` 형식을 순서대로 읽습니다. 무료 제공량은 서비스별로 자주 바뀌므로 상용화 전에는 요금제와 사용 제한을 다시 확인해야 합니다.
